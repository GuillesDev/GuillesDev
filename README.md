# Guillermo López

Llevo más de 5 años haciendo grafismo y motion en directo para televisión (Cuatro, Telecinco, Mediaset, Telefónica). Por el camino aprendí a automatizar lo repetitivo y a construir webs a medida, así que ahora hago las tres cosas.

**Portfolio y reel:** https://portfolio-guilles.vercel.app

**Black Gum.** Una web full-stack que monté de cero para una productora: la parte pública y un panel de administración propio con subida de vídeos por trozos. De ahí saqué este patrón y lo dejé en abierto, ya limpio: [resumable-chunked-upload](https://github.com/GuillesDev/resumable-chunked-upload).

Madrid o Sevilla, también remoto.

Trabajo con TypeScript, Next.js, Astro, Three.js y Prisma.
